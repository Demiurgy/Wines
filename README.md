### визуальный код зашитый в этикетки по чему-то в вине
1) игристое - модель очень хорошо научилась определять из датасета игристое - файл sparkling_visual_report.zip
В переводе на человеческий
1. Этикетка выглядит “праздничнее” и богаче Больше декоративных элементов, “фактурности”, иногда медальки/печати/орнаменты.
2. Больше цветных акцентов, но не обязательно “ярче” Скорее акценты (ленты, герб, плашки), чем кричащая палитра. Часто это “премиальные” тона: золото/беж/тёплые оттенки.
3. Чёткие тёмные элементы на фоне Надписи/рамки/контуры заметнее. Это делает этикетку “официальной” и читаемой.
4. Чуть меньше ощущения “мелкой каши” текста В среднем игристое чаще выглядит более “собранно”: заметные элементы, меньше мелких плотных деталей по всей площади (но бывают исключения).
Исключения
Есть бренды/линейки, которые делают игристое в стиле тихого:
* минималистично,
* почти без акцентов,
* или наоборот “барочно” и так у них выглядят вообще все вина.
(Как дилетант в дизайне я ХЗ как это знание применить к реальности :) )
